Android 接入验证

tip

请确保Android母包可以游戏正常运行。激励视频可以正常获得奖励。
日志Tag: TKG_DEBUG_LOG

新版本Android验证方式(V2.2.9.1+)​

Android母包正确集成SDK后，插屏和激励视频可弹出测试广告，Banner和Native在调用时机会弹出相应Tips。

1. 激励视频​

调用激励视频可正常展示测试广告，观看完成可正常获得奖励，观看中途关闭不可获得奖励。

2. 插屏​

调用插屏可正常展示测试广告，关闭插屏后游戏可正常继续。（插屏之间有时间间隔限制 eg.30s，达到时间间隔可展示插屏广告）
