iOS 接入验证

重要！

Xcode工程自测说明:
在SDK接入完成后，可使用您的开发者账号下任意包名对接入游戏进行测试，如需开发者账号支持，还请与我们联系。

AppTrackingTransparency 弹框​

请确保有 AppTrackingTransparency 弹框弹出，SDK内已处理，无需进行特殊操作。

集成SDK后，插屏和激励视频可弹出测试广告，Banner和Native在调用时机会弹出相应Tips。

激励视频​

调用激励视频可正常展示测试广告，关闭后获得激励回调。

插屏​

调用插屏可正常展示测试广告，关闭插屏后游戏可正常继续。
