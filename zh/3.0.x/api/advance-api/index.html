1 游戏失去/获得焦点​

APIParameterDescription
SetGameFocusListener(Action<bool> _gameFocusAction)设置游戏失去焦点回调
true:获得焦点
false:失去焦点

每次开始播广告等情况会被触发，可统一处理暂停游戏声音等。

1.1 示例代码​

TKGSDKManager.Instance.SetGameFocusListener((_isFocus) => {
    if (_isFocus)
    {
        Debug.Log("set game focus");
    }
    else
    {
        Debug.Log("lose game focus");
    }
});

5 获取当前用户来源​

APIParameterDescription
SetUserSourceListener(Action <bool, string> _userSourceAction)设置当前用户来源回调监听
_isOrigin: true 自然用户 false 非自然用户
该接口为异步返回。如一次游戏内多次调用该接口情况，只首次有返回值。

TKGSDKManager.Instance.SetUserSourceListener((_isOrigin, _source) =>
{
    if (_isOrigin)
    {
        Debug.Log("is origin user");
    }
    else
    {
        Debug.Log("is not origin user");
    }
});

6 SDK通用回调接口​

APIParameterDescription
SetTKGCommonCallback(Action<CommonCallbackCode, string> _commonCallback)
SDK统一事件回调接口。
用CommonCallbackCode来进行区分事件名称。 FirebaseOnlineParamGet_Succ: firebase在线参数获取成功。 FirebaseOnlineParamGet_Failed: firebase在线参数获取失败。

});

7 自定义在线参数​

APIParameterDescription
GetConfigStr(TKGParamKey _key)获取string类型的在线参数。
需事先在TKGParams中定义在线参数key及默认值
GetConfigInt(TKGParamKey _key)获取int类型的在线参数。
需事先在TKGParams中定义在线参数key及默认值
GetConfigBool(TKGParamKey _key)获取bool类型的在线参数。
需事先在TKGParams中定义在线参数key及默认值
true: 默认值配1; false: 默认值配0

7.1 示例代码​

Asset/TKGSDK/Config/Scripts/TKGParams.cs
// set key
public enum TKGParamKey
{
    RemoveAdsShowCount,
    LevelList,
    HasBlock,
    .
    .
}

// add key & default value
public static class TKGParams
{
    public static readonly Dictionary<string, object> OnlineParamDic = new Dictionary<string, object>()
    {
        // key - default value
        { TKGParamKey.RemoveAdsShowCount.ToString(), 4 },    // int value    // 
        { TKGParamKey.LevelList.ToString(), "1,78,72,70,64,65,4,66" }, // string value
        { TKGParamKey.HasBlock.ToString(), 0 },     // bool value // 1:true 0:false
    };
}

// get online param value
int onlineIntValue = TKGSDKManager.Instance.GetConfigInt(TKGParamKey.RemoveAdsShowCount);
string onlineStrValue = TKGSDKManager.Instance.GetConfigStr(TKGParamKey.LevelList);
bool onlineBoolValue = TKGSDKManager.Instance.GetConfigBool(TKGParamKey.HasBlock);

10 震动​

APIParameterDescription
Shake(int _shakeType, float _intensity = 1)触发手机震动

_shakeType 震动强弱:
0:轻微
1:中度
2:重度

_intensity 震动持续时间:
0 ~ 1

11 推送​

APIParameterDescription
RegistAPNS-调用后弹出系统推送授权弹框
RegistNotification(string notiId, string body, string fireDate, int badge = 1, string title = "", string subTitle = "")注册推送

notiId: 注册推送的ID，具有唯一性

body: 推送内容，非空，否则注册不成功

fireDate: 推送内容触发时间，format 为 ：yyyy-MM-dd HH:mm:ss；需大于当前时间，否则注册不成功

title: 推送标题

subTitle: 推送副标题

RemoveNotification(string notiId)根据推送ID移除该条推送
RemoveNotification-移除全部推送

11.1 示例代码​

private void Awake()
{
    TKGSDKManager.Instance.RegistAPNS();
}

private void OnApplicationFocus(bool isFocus)
{
    TKGSDKManager.Instance.RemoveAllNotifications();

    if (!isFocus)
    {
        TKGSDKManager.Instance.RegistNotification("notificationID","hello world", now.AddDays(1).ToString("yyyy-MM-dd HH:mm:ss"),1,"Title","SubTitle");
    }
}

13 来源控制 (v2.2.9新增)​

APIParameterDescription
SetFunctionSwitchListener(List<FunctionType> _functionKeys, Action<FunctionType, bool> _functionSwitchCallback)List<FunctionType> _functionKeys, Action<FunctionType, bool> _functionSwitchCallback)设置各功能回调监听
_functionKeys: 功能名称集合
_functionSwitchCallback: 功能集合回调

13.1 示例代码​


void Start() 
{
    TKGSDKManager.Instance.SetFunctionSwitchListener(new List<FunctionType>() 
    { 
        FunctionType.Function_wangz, 
        FunctionType.Function_bing 
    
    }, OnFunsSwitchCallback);

}


/**
    _type : 功能名称
    _result : 是否开启当前功能,true:开启，false:不开启
*/
private void OnFunsSwitchCallback(FunctionType _type, bool _result)
{
    Debug.Log("[ToukaSDKDemo]：FunctionType:" + _type +"result:"+ _result);
}

说明：
1. 该接口为异步返回。List的长度决定回调方法执行的次数；如list的长度为2，则会执行两次回调方法。
2. 此方法需要在初始化SDK后调用。
