内购

danger
  • 需要添加 USE_IAP 宏定义
  • Unity In App Purchasing 插件版本需高于4.4.0
  • 如果项目中有使用到 Unity Gaming Services 组件时，确保其手动初始化，且初始化SDK方法需放在Unity Gaming Services 组件初始化成功回调中执行

内购接入方法​

1、添加宏定义​

USE_IAP

2、在 IAPProducts.cs 文件中添加商品ID​

public const string NoAds = "test.tremoveads";

public static readonly Dictionary<string, ProductType> ProductDic = new Dictionary<string, ProductType>()
{
    // ProductType.NonConsumable  只能购买一次。适合一次性购买的商品，如额外的关卡
    // ProductType.Consumable     可以反复购买。适合虚拟货币等可消耗商品
    // ProductType.Subscription   可以反复购买和恢复。耐用品，但有效期有限（订阅）
    
    { NoAds, ProductType.NonConsumable}
        
};

3、添加内购回调​


IAPTool.Instance.OnPurchaseDone += OnPurchaseResult;
private void OnPurchaseResult(string pID, bool pResult)
{
    if (pID.Equals(IAPProducts.NoAds) && pResult)
    {
        // 内购成功后，去除广告
        TKGSDKManager.Instance.IsRemoveAds = true;
    }
}

4、内购接口​

IAPTool.Instance.BuyProductByID(IAPProducts.NoAds);

内购事件​

内购按钮展示​

void LogIAPBtnShow(string _iap, string _id, string _currency, string _price)

NameisRequiredTypeDescription
iapYstring游戏侧自定义商品名称
idYstring商品ID
GP：ProductID
iOS：SKU
currencyYstring货币单位,如：USD
priceYstring购买价格

调用时机： 内购按钮展示时调用。 

TKGSDKManager.Instance.LogIAPBtnShow("200钻石","com.game.diamond200","USD","1.99");

内购按钮点击​

void LogIAPBtnClick(string _iap, string _id, string _currency, string _price) (参数同上)

调用时机： 内购按钮点击时调用。 

TKGSDKManager.Instance.LogIAPBtnClick("200钻石","com.game.diamond200","USD","1.99");

首次购买成功​

void LogIAPFirstPurchase(string _iap, string _id, string _currency, string _price) (参数同上)

调用时机： 首次购买成功时上报

TKGSDKManager.Instance.LogIAPFirstPurchase("200钻石","com.game.diamond200","USD","1.99");

购买成功​

void LogIAPSuccess(string _iap, string _id, string _gameOrderid, string _paymentChannelOrderid, string _productToken, string _paymentMethod, string _currency, string _price)

NameisRequiredTypeDescription
iapYstring游戏侧自定义商品名称
idYstring商品ID
GooglePlay：ProductID
iOS：SKU
gameOrderidYstring游戏侧订单id（没有可传空字符串 ""）
paymentChannelOrderidYstringGooglePlay/ iOS 上的财务交易。每当发生财务交易时，系统都会创建订单ID。在IAPTool.cs文件 public PurchaseProcessingResult ProcessPurchase(PurchaseEventArgs e)方法中获取到的orderId的值。
productTokenYstringGooglePlay上购买成功时，会生成一个购买令牌作为唯一标识，其他支付方式传空即可。在IAPTool.cs文件 public PurchaseProcessingResult ProcessPurchase(PurchaseEventArgs e)方法中获取到的purchaseToken的值。
paymentMethodYstring官方支付渠道传：AppStore、GooglePlay即可，其他支付渠道传对应渠道名称
currencyYstring货币单位,如：USD
priceYstring购买价格

调用时机： 购买成功时上报

TKGSDKManager.Instance.LogIAPSuccess("200钻石","com.game.diamond200","game-20230712035656546","GPA.3327-1005-9772-34199","fhlkemmlgnoaedkjmlobdkce.AO-J1OxTtpuomZK4pjutaKDyOqQfFBF122_NgIlMzU_3xrYdvvv9ed1Y-WZVC7yXBVQDKKjx25rs8S02mHjE9GUmfhXbDT_TIyYHQCEyJbTmdnS82N5FR4o","GooglePlay","USD","1.99");
