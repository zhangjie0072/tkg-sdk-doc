Version: 3.1.x

FAQ

Unity​

1、移除 Package Manager 中 Mobile Notifications 插件。​

此插件会导致iOS15系统第二次启动时无法进入游戏。

2、移除 Package Manager 中 Advertisement 插件​

Touks SDK 中已经包含了 UnityAds，插件中也集成会导致导出后的xcode工程有文件冲突。

iOS​

1、Xcode报错：Legacy Build System​

当你编译Xcode时，看到如下报错，可参考如下方式解决。

  1. 选择File > Project Settings

  1. 在Shared Project Settings中, 修改Build System settings为New Build System。

2、Error Undefined symbol:__swift_FORCE_LOAD_swift CompatibilityConcurrency​

错误：

解决方案： Xcode 13.3.1 或更高版本

Android​

1、Target API Level Error​

请根据 构建版本->构建Android平台->Minimun API Level : API Level 22

2、Execution failed for task ':launcher:checkReleaseDuplicateClasses​

请确认如果是更新SDK版本导致的问题，那需要先将老的SDK版本删除，并且要导出新的Android Studio工程。而不是覆盖导出到老版本SDK导出的工程。
请确认 构建版本->构建Android平台->Custom Main Manifest配置正确。
