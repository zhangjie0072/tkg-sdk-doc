3 Advertising integration‚Äã

3.1 InterstitialAd‚Äã

APIParameterDescription
ShowInterstitialAd(TKGIVAdPositionName _adPos, Action _callback = null, IVADType _IvType = IVADType.IV1)Show Interstitial Ad

_adPos: Name of interstitial ad placement.ÔºàPlease define the TKGIVAdPositionName firstÔºâ
_callback: Callback of ad close and playback fail.
_IvType: Type of frequency control
IsReadyInterstitialAd-Whether InterstitialAd can be displayed.
General games do not need to call by themselves.

3.1.1 Sample code‚Äã

// show interstitial ad
TKGSDKManager.Instance.ShowInterstitialAd(TKGIVAdPositionName.IV_Retry);

// is ready interstitial ad
bool isReadyIV = TKGSDKManager.Instance.IsReadyInterstitialAd();

3.2 RewardAd‚Äã

APIParameterDescription
ShowRewardAd(TKGRVPositionName _adPos, Action<bool> _rewardCallback = null, Action _showFailedCallback = null, bool _showSDKToast = false)Show RewardAd

_adPos: Name of reward ad placement.ÔºàPlease define the TKGRVPositionName first.Ôºâ
_rewardCallback: Callback for reward.(Recommended developers issue rewards in this callback.)
_showFailedCallback: Callback of ad close and playback fail.
_showSDKToast: Whether to use the SDK to prompt the video presentation failure page. true:Use the failure page provided by the SDK. false:Do not use the failure notification page provided by the SDK.
IsReadyRewardAd-Whether RewardAd can be displayed.
General games do not need to call by themselves.

3.2.1 Sample code‚Äã

// show reward ad
TKGSDKManager.Instance.ShowRewardAd(TKGRVPositionName.RV_GetDoubleCoin, RewardResult);

private void RewardResult(bool _bReward)
{
    Debug.Log("[ToukaSDKDemo] Reward resultÔºö" + _bReward);
}

// is ready reward ad
bool isReadyRV = TKGSDKManager.Instance. IsReadyRewardAd();

3.4 Native ad‚Äã

APIParameterDescription
ShowNative(RectTransform pRect, Camera pCam = null, string pAdPos = "")Show Native Ad

pRect: Native advertising display location
pCam: If the UI uses camera mode, you must use an orthogonal camera
pAdPos: display native AD point name
IsReadyNativeAd-Whether NativeAd can be displayed.
General games do not need to call by themselves.

4 Event Tracking‚Äã

4.1 Level Progression Events‚Äã

APIParameterDescription
LevelStart(int _level)
(string _level)		Level Start
_level: current level id.
LevelEnd(int _level, StageResult _stageResult)
(string _level, StageResult _stageResult)		Level End
_level: current level id.
_stageResult: level result

StageResult : StageSucc / StageFail / retry in level: StageRetry / back in level: StageBack

4.1.1 LevelStart‚Äã

TKGSDKManager.Instance.LevelStart(1);
TKGSDKManager.Instance.LevelStart("S_1");

4.1.2 LevelEnd‚Äã

TKGSDKManager.Instance.LevelEnd(1, StageResult.StageSucc);
TKGSDKManager.Instance.LevelEnd("S_1", StageResult.StageFail);
TKGSDKManager.Instance.LevelEnd(3, StageResult.StageRetry);
TKGSDKManager.Instance.LevelEnd("S_2", StageResult.StageBack);

5 Purchase (v3.0.4 new)‚Äã

5.1 Macro:‚Äã

Add macro definition: USE_IAP

5.2 Product ID:‚Äã

Add the product ID to the IAPProducts.cs file:

public const string NoAds = "test.tremoveads";

public static readonly Dictionary<string, ProductType> ProductDic = new Dictionary<string, ProductType>()
{
    // ProductType.NonConsumable  only buy once. Good for one-time purchases, such as extra levels
    // ProductType.Consumable     can be purchased repeatedly. Suitable for consumable goods such as virtual currency
    // ProductType.Subscription   can be purchased and restored repeatedly. Durable goods, but for a limited period (subscription)
    
    { NoAds, ProductType.NonConsumable}
        
};

5.3 Add purchase callback:‚Äã


IAPTool.Instance.OnPurchaseDone += OnPurchaseResult;
private void OnPurchaseResult(string pID, bool pResult)
{
    if (pID.Equals(IAPProducts.NoAds) && pResult)
    {
        // After successful internal purchase, remove advertising.
        TKGSDKManager.Instance.IsRemoveAds = true;
    }
}
