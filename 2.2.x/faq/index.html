FAQ

Unity​

1、Remove the 'Mobile Notifications' plugin from Package Manager.​

This plugin will cause iOS15 to fail to enter the game on the second boot.

2、Remove the 'Advertisement' plugin from Package Manager.​

Touks SDK has included UnityAds, and integration in the plugin will lead to file conflicts in the exported Xcode project.

iOS​

1. Legacy Build System Error on Xcode​

When exporting the project to Xcode, if you see a building error saying, "The Legacy Build System will be removed in a future release. You can configure the selected build system and this deprecation message in File > Project Settings" please do the following to resolve this issue.

Step1. Navigate to File, then select Project Settings

Step2. Under Shared Project Settings, change the Build System settings to New Build System

2. Library not loaded: ....KSAdSDK.... Error on Xcode​

If you see a building error about KSAdSDK similar to the screenshot below. Please do the Add dynamic library for ads sdk steps to resolve this issue.

3. Required parameter is empty​

InitSDK will check the necessary parameters. If the following error is reported, please check the parameter configuration.

Android​

1、Target API Level Error​

Please Build Package -> For Android ->Minimun API Level : API Level 22

2、Execution failed for task ':launcher:checkReleaseDuplicateClasses​

If the problem is caused by updating new SDK version, please delete the old version first, and export new Android Studio project. don't append to old version's exported project.
Please make sure Build Package->For Android->Custom Main Manifest config.
