Advanced Integration

1. Game focus changed callback​

TKGSDKManager.Instance.SetGameFocusListener((_isFocus) => {
    if (_isFocus)
    {
        Debug.Log("set game focus");
    }
    else
    {
        Debug.Log("lose game focus");
    }
});

When the ad is displayed, your game is going to lose focus . This method allows you to receive focus changes.

API: 

void SetGameFocusListener(Action<bool> _gameFocusAction)
NameisRequiredTypeDescription
_gameFocusActionYAction<bool>Pass your callback receiver method.
Your method needs to implement one parameter to receive a boolean value.

2. Ratings and reviews​

Attention: only available on iOS at the moment!

TKGSDKManager.Instance.Review();

Call this method wherever needed Presenting the user with a request for an App Store review using Review() to get feedback on your game.

TKGSDKManager.Instance.OpenMoreGame();

Open the store's own games listing page. The URL key in the configuration file is tka_url_moregame

7. Custom Remote Config​

Get Remote Config​

  • Make changes to your game dynamic by defining remote configs.
  • Before you call any of the methods to get the remote parameters, you must pre-define in the config file.

Pre define step: step1. Declare an enum for all of your param keys:

Asset/TKGSDK/Config/Scripts/TKGParams.cs
public enum TKGParamKey
{
    RemoveAdsShowCount,
    LevelList,
    HasBlock,
    .
    .
}

step2. Add your enum into OnlineParamDic:

The default values of all customized online parameters are defined here.
Asset/TKGSDK/Config/Scripts/TKGParams.cs
  
public static class TKGParams
{
    public static readonly Dictionary<string, object> OnlineParamDic = new Dictionary<string, object>()
    {
        // key - default value
        { TKGParamKey.RemoveAdsShowCount.ToString(), 4 },    // int value    // 
        { TKGParamKey.LevelList.ToString(), "1,78,72,70,64,65,4,66" }, // string value
        { TKGParamKey.HasBlock.ToString(), 0 },     // bool value // 1:true 0:false
    };
}

step3. Get remote parameters in your game refer to the method list.

// get online param value
int onlineIntValue = TKGSDKManager.Instance.GetConfigInt(TKGParamKey.RemoveAdsShowCount);
string onlineStrValue = TKGSDKManager.Instance.GetConfigStr(TKGParamKey.LevelList);
bool onlineBoolValue = TKGSDKManager.Instance.GetConfigBool(TKGParamKey.HasBlock);

API: 

int GetConfigInt(TKGParamKey _key);    
string GetConfigStr(TKGParamKey _key);  
bool GetConfigBool(TKGParamKey _key);
NameisRequiredTypeDescription
_keyYTKGParamKeyOnline param key
Defined in the your game product requirement.

10. Notification （iOS Only）​

1. Request notification permission​

TKGSDKManager.Instance.RegistAPNS();

After the call, the system authorization notification box will pop up.

2. Registered notification​

TKGSDKManager.Instance.RegistNotification("notificationID","hello world", now.AddDays(1).ToString("yyyy-MM-dd HH:mm:ss"),1,"Title","SubTitle");

API:

void RegistNotification(string notiId, string body, string fireDate, int badge = 1, string title = "", string subTitle = "")

NameisRequiredTypeDescription
notiIdYstringThe ID of the registration notification is unique.
bodyYstringNotification content, not empty, otherwise the registration is unsuccessful.
fireDateYstringThe format is yyyy-MM-dd HH:mm:ss. The value is greater than the current time; otherwise, the registration fails.
titleNstringNotification title.
subTitleNstringNotification subtitle.

3. Remove a specified notification​

TKGSDKManager.Instance.RemoveNotification("notificationID");

API:

void RemoveNotification(string notiId)<br/>
NameisRequiredTypeDescription
notiIdYstringRemove the notification according to the notification ID.

4. Remove all notifications​

TKGSDKManager.Instance.RemoveAllNotifications();

Remove all notifications after the call.

12、Source control （v2.2.9 new）​

Note：
1. This interface returns asynchronously. The length of List determines the number of times the callback method is executed; If the length of the list is 2, the callback method is executed twice.

void Start() 
{
  TKGSDKManager.Instance.SetFunctionSwitchListener(new List<FunctionType>() { FunctionType.Function_wangz, FunctionType.Function_bing }, OnFunsSwitchCallback);

}

private void OnFunsSwitchCallback(FunctionType _type, bool _result)
{
  Debug.Log("[ToukaSDKDemo]：FunctionType:" + _type +"result:"+ _result);
}

API: 

void SetFunctionSwitchListener(List<FunctionType> _functionKeys, Action<FunctionType, bool> _functionSwitchCallback);
NameisRequiredTypeDescription
_functionKeysYList<FunctionType>
Set of function names
_functionSwitchCallbackYAction<FunctionType, bool>The callback event returns the name of each function and the corresponding switch.
FunctionType: Function type
bool: Whether to enable the current function, true - open / false - not open
