7. Custom Remote Config‚Äã

APIParameterDescription
GetConfigStr(TKGParamKey _key)get string type remote config.
Need define key and default value first in TKGParams.
GetConfigInt(TKGParamKey _key)get int type remote config.
Need define key and default value first in TKGParams.
GetConfigBool(TKGParamKey _key)get bool type remote config.
Need define key and default value first in TKGParams.
true: set default value to 1; false: set default value to 0

7.1 Sample code‚Äã

Asset/TKGSDK/Config/Scripts/TKGParams.cs
// set key
public enum TKGParamKey
{
    RemoveAdsShowCount,
    LevelList,
    HasBlock,
    .
    .
}

// add key & default value
public static class TKGParams
{
    public static readonly Dictionary<string, object> OnlineParamDic = new Dictionary<string, object>()
    {
        // key - default value
        { TKGParamKey.RemoveAdsShowCount.ToString(), 4 },    // int value    // 
        { TKGParamKey.LevelList.ToString(), "1,78,72,70,64,65,4,66" }, // string value
        { TKGParamKey.HasBlock.ToString(), 0 },     // bool value // 1:true 0:false
    };
}

// get online param value
int onlineIntValue = TKGSDKManager.Instance.GetConfigInt(TKGParamKey.RemoveAdsShowCount);
string onlineStrValue = TKGSDKManager.Instance.GetConfigStr(TKGParamKey.LevelList);
bool onlineBoolValue = TKGSDKManager.Instance.GetConfigBool(TKGParamKey.HasBlock);

11. Notification‚Äã

APIParameterDescription
RegistAPNS-After the call, the system authorization notification box will pop up.
RegistNotification(string notiId, string body, string fireDate, int badge = 1, string title = "", string subTitle = "")Registered notification

notiId: The ID of the registration notification is unique.

body: Notification content, not empty, otherwise the registration is unsuccessful.

fireDate: The format is yyyy-MM-dd HH:mm:ss. The value is greater than the current time; otherwise, the registration fails.

title: Notification title

subTitle: Notification subtitle

RemoveNotification(string notiId)Remove the notification according to the notification ID.
RemoveNotification-Remove all notifications after the call.

10.1 Sample code‚Äã

private void Awake()
{
    TKGSDKManager.Instance.RegistAPNS();
}

private void OnApplicationFocus(bool isFocus)
{
    TKGSDKManager.Instance.RemoveAllNotifications();

    if (!isFocus)
    {
        TKGSDKManager.Instance.RegistNotification("notificationID","hello world", now.AddDays(1).ToString("yyyy-MM-dd HH:mm:ss"),1,"Title","SubTitle");
    }
}
